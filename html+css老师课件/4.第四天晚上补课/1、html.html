2、半闭合标签：单独出现的，不可以嵌套 html标签按自身属性分类： 1、块级元素： block 1）独占一行 2）可以设置宽度和高度 3）在不设置宽度和高度的情况下，宽度是父级元素的宽度，高度是内容的高度 div、p、h1~h6、ul、ol、li、dl、dt、dd、table、tr、td、form、pre、hr

2、行内元素：inline 1）不独占一行，在一行显示 2）不可以设置宽度和高度 3）在不设置宽度和高度的情况下，宽度是自身内容宽度，高度是自身内容的高度 4) 浏览器可以识别行内元素之间的空格 span、i、em、b、strong、u、s、del、sup、sub、br、a 行内元素 斜体 3、行内块级元素：inline-block 1）不独占一行 2）可以设置宽度和高度 3）在不设置宽度和高度的情况下，宽度是自身内容宽度，高度是自身内容的高度 4）浏览器可以识别行内块级元素之间的空格 img、input 标签语义化：合理的标签做合理的事 无意义 划分结构：div 链接 a 标题 hx logo 网站的标题 h1（在一个网页中只出现一次） 一般用 h3、h4、h5 段落 p 强制换行br 结构或格式相同，并且重复出现的内容 ul、ol、li dl、dt、dd 图片 img 文本框 input 几个文字 span i b 小图标 span i b s u 尽量不用 em、strong（加强语气的作用）
